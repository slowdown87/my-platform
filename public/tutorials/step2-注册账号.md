## 目标

注册 4 个免费云服务账号。这些账号让您的网站能够存储代码、托管网站、存储数据、使用AI。

---

## 1️⃣ 注册 GitHub（代码托管）

GitHub 是全世界最大的代码托管平台，您的代码会存在这里。

**操作步骤：**
1. 打开浏览器，访问：**https://github.com**
2. 点击右上角 **"Sign up"**（注册）
3. 输入您的 **邮箱**（建议用常用邮箱）
4. 设置一个 **密码**（记住它！）
5. 设置一个 **用户名**（英文，比如 `myname123`，这个会出现在您的网址里）
6. 完成验证（可能需要做一个小拼图游戏）
7. 去邮箱里点击验证邮件

**💡 小提示：**
- 用户名只能用英文字母、数字和连字符
- 用户名一旦设置不能改，想好再填
- 密码要包含大小写字母、数字和特殊符号

**✅ 验证：**
登录 GitHub 后能看到您的头像和用户名就说明成功了。

---

## 2️⃣ 注册 Netlify（网站托管）

Netlify 会把您的网站部署到全球服务器，让任何人都能访问。**国内可以正常访问部署的网站**。

**操作步骤：**
1. 打开浏览器，访问：**https://app.netlify.com**
2. 点击右上角 **"Sign up"**（注册）
3. 选择 **"Continue with GitHub"**（用刚注册的 GitHub 账号直接登录，最方便！）
4. 授权 Netlify 访问您的 GitHub
5. 可能需要填写一些基本信息，按提示完成即可

**💡 为什么选 Netlify 而不是 Vercel？**
- Vercel 部署的网站在国内**打不开**
- Netlify 部署的网站在国内**可以正常访问**

**✅ 验证：**
能看到 Netlify 的控制面板（Dashboard）就说明成功了。

---

## 3️⃣ 注册 Supabase（数据库 + 用户认证）

Supabase 提供数据库和用户登录功能，是您网站的核心后端。

**操作步骤：**
1. 打开浏览器，访问：**https://supabase.com**
2. 点击右上角 **"Start your project"**
3. 选择 **"Continue with GitHub"**（用 GitHub 账号直接登录，推荐！）
4. 授权 Supabase 访问您的 GitHub
5. Supabase 会要求您设置一个组织名称（随便填，比如 `my-organization`）

**✅ 验证：**
能看到 Supabase 的控制面板就说明成功了。

---

## 4️⃣ 注册阿里百炼（AI 对话功能）

阿里百炼提供免费的 AI 对话能力，国内直连。

**操作步骤：**
1. 打开浏览器，访问：**https://bailian.console.aliyun.com**
2. 用阿里云账号登录（如果没有，用手机号注册一个）
3. 完成实名认证（按提示用手机号验证）
4. 点击左侧菜单 **"API-KEY管理"**
5. 点击 **"创建 API Key"**
6. 复制生成的 Key（以 `sk-` 开头），保存在记事本里
7. 在 **"模型广场"** 中开通免费模型（推荐 `qwen-turbo` 或 `qwen2.5-omni-7b`）

**⚠️ 重要：**
- API Key 要保密！不要告诉别人
- 免费模型每月 100 万 tokens，个人使用完全够

**✅ 验证：**
您已经复制了 API Key 并开通了免费模型就说明成功了。
