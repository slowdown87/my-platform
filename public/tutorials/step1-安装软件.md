## 目标

安装 3 个必备软件，这些是做任何编程项目都需要的基础工具，装一次以后一直能用。

---

## 1️⃣ 安装 VS Code（代码编辑器）

这就是我们写代码的地方，相当于"写代码的纸和笔"。

**操作步骤：**
1. 打开浏览器，访问：**https://code.visualstudio.com**
2. 点击页面上的 **大蓝色按钮** "Download for Windows"
3. 下载完成后，**双击** 下载的安装文件
4. 安装过程中**一路点"下一步"**就行，所有选项保持默认
5. 安装完成后，桌面会出现一个蓝色的 VS Code 图标

**✅ 验证方法：**
双击桌面上的 VS Code 图标，能打开就说明安装成功。

---

## 2️⃣ 安装 Node.js（JavaScript 运行环境）

这个让您的电脑能"读懂"和"运行" JavaScript 代码。

**操作步骤：**
1. 打开浏览器，访问：**https://nodejs.org**
2. 页面上会看到两个按钮，点击左边那个 **"LTS"**（长期支持版）
3. 下载完成后，**双击** 安装文件
4. 安装过程中**一路点"下一步"**，保持默认选项

**✅ 验证方法：**
- 按 `Win + R` 键，输入 `cmd`，按回车，打开命令行窗口
- 输入：`node -v` 然后按回车
- 如果显示类似 `v20.x.x` 的版本号，就说明安装成功了！
- 再输入：`npm -v` 按回车，也应该显示版本号

---

## 3️⃣ 安装 Git（代码版本管理）

这个用来把代码上传到 GitHub，实现自动部署。

**操作步骤：**
1. 打开浏览器，访问：**https://git-scm.com/download/win**
2. 下载完成后，**双击** 安装文件
3. 安装过程中**一路点"下一步"**，保持默认选项
4. 如果看到 "Choosing the default editor" 页面，选择 **Visual Studio Code**

**✅ 验证方法：**
- 打开命令行窗口（Win + R → 输入 cmd → 回车）
- 输入：`git --version` 按回车
- 如果显示类似 `git version 2.x.x`，就说明安装成功了！

## 常见问题

**Q: 这些软件会不会有病毒？**
A: 不会。VS Code、Node.js、Git 都是全世界程序员都在用的官方软件，非常安全。请只从官方网站下载。

**Q: 为什么要装这三个软件？**
A:
- **VS Code**：写代码的地方，就像用 Word 写文档一样
- **Node.js**：让电脑能运行 JavaScript 代码
- **Git**：把代码传到网上，这样别人就能访问您的网站了
